## Background

- Define um fundo para o nosso elemento
- Sua area de atuação é a caixa toda
- por padrao, é transparente

### exemplos

- Usar cores solidas
- Usar imagens
- Controlar
    - A posição das imagens,
    - se elas se repetem ou nao
    - o tamanho delas na caixa
- Usar cor e imagem juntas
- Usar cor gradiente
