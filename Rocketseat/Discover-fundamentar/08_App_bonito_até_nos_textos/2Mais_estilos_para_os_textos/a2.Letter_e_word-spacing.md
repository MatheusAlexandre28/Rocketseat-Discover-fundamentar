## letter-spacing

* Espaços entre caracteres

'''css
p {
    letter-spacing: 4px;
}
'''

https://developer.mozilla.org/en-US/docs/Web/CSS/letter-spacing

## Word spacing

* Define o espaçamento entre palavras

'''css
p {
    word-spacing: 1em;
}
'''
