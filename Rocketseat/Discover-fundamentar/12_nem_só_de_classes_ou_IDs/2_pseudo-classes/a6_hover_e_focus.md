## Ações do usuario

* :hover 

passas o mouse e muda de cor use este codigo:

'''HTML
<a href="#">Clique aqui</a>
'''
'''CSS
a:hover {
  color: red;
}
'''


* :focus

muda a cor da borda quando um elemento esta focado tipo um <input>, veja nesse codigo.. com o CSS e sem o CSS:

'''HTML
<input type="text">
'''

'''CSS
input:focus {
  border-color: blue;
  outline: none;
}
'''
